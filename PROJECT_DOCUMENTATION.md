# ObserveX - AI Home Security Landing Page

## Project Overview

ObserveX is a premium, investor-ready landing page for an AI-powered intelligent home security and behavioral threat detection system. The website showcases cutting-edge security technology with a focus on proactive threat prevention rather than reactive monitoring.

## Design Philosophy: Glassmorphic AI Intelligence

The website employs a sophisticated design language combining glassmorphism with AI visualization aesthetics:

- **Visual Foundation:** Deep navy backgrounds (#0f172a) with layered transparency effects
- **Color Palette:** Cyan (#06b6d4), purple (#a78bfa), and emerald (#10b981) accents on dark backgrounds
- **Typography:** Sora (headlines) + JetBrains Mono (technical elements) + Inter (body text)
- **Animation Framework:** Framer Motion for scroll-triggered reveals, hover effects, and continuous background animations
- **Aesthetic:** Futuristic, premium, cinematic—designed to feel like a well-funded startup

## Color System (OKLCH Format)

```
Primary Background: oklch(0.08 0.02 240)      // Deep navy
Secondary Background: oklch(0.12 0.03 245)    // Dark blue
Cyan Accent: oklch(0.65 0.18 200)             // Bright cyan
Purple Accent: oklch(0.65 0.15 280)           // Soft purple
Emerald Accent: oklch(0.60 0.15 160)          // Emerald green
Text Primary: oklch(1 0 0)                    // White
Text Secondary: oklch(0.88 0.02 240)          // Light gray
```

## Animation Guidelines

- **Entrance Animations:** Fade in with slight upward motion (300-500ms)
- **Scroll Triggers:** Sections reveal as they enter viewport
- **Hover Effects:** Subtle color shifts, shadow deepening, glow intensification
- **Continuous Animations:** Floating particles, gradient shifts (6-10s duration)
- **Loading:** Circular scan animation with rotating rings

## Responsive Breakpoints

- **Mobile:** < 640px (full-width, stacked layouts)
- **Tablet:** 640px - 1024px (2-column grids)
- **Desktop:** > 1024px (full multi-column layouts)

## Design Decisions

### Why Glassmorphism?
Glassmorphism creates visual depth and suggests layered AI processing happening in the background. It's modern, premium, and aligns with top-tier SaaS companies like Notion and Linear.

### Why Dark Theme?
Dark themes are associated with technology, sophistication, and premium products. They also reduce eye strain and feel more "futuristic"—perfect for an AI security product.

### Why Cyan + Purple?
Cyan represents technology and trust (commonly used in security/tech). Purple adds sophistication and creativity. Together, they create visual interest without being overwhelming.

### Why Sora Font?
Sora is a contemporary, geometric sans-serif that feels modern and tech-forward. It's more distinctive than Inter while remaining highly readable.
